Premerie league Manager


Match Section

The Match section displays all the matches played between clubs in the Premeir league system. Users can search for a match by using the match date.


Club Section

All the clubs entered in Premier league manager are shown in this section. Their wins, defeats, draws, points, and all other statistics are also shown in this section. User can sort clubs according to the wins, points and goals scored.


Controllers

Users can navigate between pages by using links given in the header.
