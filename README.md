# premier-league-manager-
How to run this project:

Go to https://start.spring.io/ and download a spring project with these properties - Maven Project, Java,2.4.2, Artifact and Name= W1790272, Jar, Java 8, Dependency=Spring reactive web
Replace its SRC folder with "premier-league-manager" src.
Keep only Main and Test folders inside the src and get other files outside from the src.
Run in intellij and edit configeration to Springboot -> ConsoleApplication.
Get the path of the "view" folder in the terminal and type "npm install". Then use "ng serve" command.
